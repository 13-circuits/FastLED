Detailed Description

These schemes are all declared as "PROGMEM", meaning that they won't take up SRAM on AVR chips until used.

Furthermore, the compiler won't even include these in your PROGMEM (flash) storage unless you specifically use each one, so you only 'pay for' those you actually use.
